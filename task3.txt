IP- Internet Protocol. Network Layers. Это числовой идентификатор устройств в сети. Используется для передачи данных между устройствами

TCP - Transmission control Protocol - изначально был создан для надежного соединения, перед тем как назначить передачу данных, протокол устанавливает соединение и потом начинает передавать данные.
Надежный протокол и не теряет пакеты информации во время передачи. Примеры протоколов которые работают по TCP. (SSH,FTP,Telnet,SMTP,Http, https)
UDP - User Data Datagram Protocol - не требует установки соединения между хостами для передачи данных. Пакеты отправляются сразу без проверки клиент и не гарантирует доставку данных и их порядок.
Скорость передачи выше чем ТСР протокол. Применяется в случаях когда важна скорось а не целостнось данных например видеостриминг или онлайн-игры итд, Примеры протоколов которые работают по UDP. (DNS,
SNMP, DHCP)
